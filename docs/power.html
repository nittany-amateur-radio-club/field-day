Power estimation

About

This page documents estimations about power usage for Field Day.

Setup

library(googlesheets4)
gs4_auth(email = params$google_email)

Import

W3TM generated a Google Sheet with data for these estimates:

https://docs.google.com/spreadsheets/d/10F2JfQ3A6oFc3VFz_GTDg_t_ZwVtrC2EGjVt0kopYQU/edit?gid=0#gid=0

if (params$update_data) {
  gs4_auth(email = params$google_email)
  power <- read_sheet(params$sheet_url)
  
  readr::write_csv(power, file = "csv/power-estimation.csv")
} else {
  message("Data not updated. Using stored copy")
}

Summarize

The modes summarized here include: FT8, ph.

Power levels estimated for the modes are as follows:

xtabs(~ mode + xmit_watts, data = power)
     xmit_watts
mode  40 100
  FT8 16   0
  ph   0  16

We’re more likely to run 40W on FT8.

Duty cycles for these modes are as follows:

xtabs(~ mode + duty_cycle, data = power)
     duty_cycle
mode  0.25 0.3 0.5
  FT8    8   0   8
  ph     8   8   0

The 0.5 duty cycle for FT8 and 0.3 duty cycle for phone assume pretty aggressive running on a frequency. More realistic duty cycles are 0.25 for phone FT8.

Operating hours are estimated as follows:

xtabs(~ op_hrs_ham, data = power)
op_hrs_ham
12 15 18 24 
 8  8  8  8

Scenario: Modest duty cycles + 12 hour operating

power |>
  dplyr::filter(op_hrs_ham == 12,
                duty_cycle == .25) |>
  dplyr::select(mode, battery_Ah, op_hrs_battery, batt_24hrs)
# A tibble: 4 × 4
  mode  battery_Ah op_hrs_battery batt_24hrs
  <chr>      <dbl>          <dbl>      <dbl>
1 FT8           30          13.0        0.92
2 FT8           40          17.4        0.69
3 ph            30           5.22       2.3 
4 ph            40           6.96       1.72

This implies that we would need 1 30 Ah battery to run FT8 for the entire 24 hr period, and 1.7 (round to 2) 40 Ah batteries to run phone the entire 24 hour period.

Scenario: Modest duty cycles + 18 hour operating

power |>
  dplyr::filter(op_hrs_ham == 18,
                duty_cycle == .25) |>
  dplyr::select(mode, battery_Ah, op_hrs_battery, batt_24hrs)
# A tibble: 4 × 4
  mode  battery_Ah op_hrs_battery batt_24hrs
  <chr>      <dbl>          <dbl>      <dbl>
1 FT8           30          13.0        1.38
2 FT8           40          17.4        1.03
3 ph            30           5.22       3.45
4 ph            40           6.96       2.59

We would need 2 40 Ah batteries for FT8 and 3 40 Ah batteries to run phone.

Scenario: Aggressive duty cycles + 24 hour operating

power |>
  dplyr::filter(op_hrs_ham == 24,
                duty_cycle > .25) |>
  dplyr::select(mode, duty_cycle, battery_Ah, op_hrs_battery, batt_24hrs)
# A tibble: 4 × 5
  mode  duty_cycle battery_Ah op_hrs_battery batt_24hrs
  <chr>      <dbl>      <dbl>          <dbl>      <dbl>
1 FT8          0.5         30           6.52       3.68
2 FT8          0.5         40           8.70       2.76
3 ph           0.3         30           4.35       5.52
4 ph           0.3         40           5.80       4.14

Under an aggressive operating and duty cycle schedule, we’d need 3 40 Ah batteries to run FT8 and 5 40 Ah batteries to operate.

Important

These scenarios do not estimate any impact of solar recharging.

However, since recharging really only helps beginning Sunday morning, the impact is only on the daylight hours on Sunday.

Batteries available

The club does not own any batteries.

W3TM has the following:

  • 1x 30 Ah
  • 2x 40 Ah

So, W3TM could cover an FT8 and a phone station for Field Day under the modest operating scenario.

For more active operating scenarios, we would need to rely on recharging batteries Sunday morning, more batteries, or larger batteries than W3TM has available.
